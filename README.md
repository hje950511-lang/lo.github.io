# 양구 체험 온 (Yanggu Experience ON) - MVP v1.0

> **"스마트폰 화면 대신 체험으로 가득한 주말을 선물하세요."**

양구군 관내의 학부모님들이 자녀를 위한 유익한 교육, 자연 생태, 문화 예술, 주말 스포츠 및 지역 봉사 프로그램을 **단 10초 만에 쉽고 빠르게 발견**할 수 있도록 돕는 모바일 퍼스트 정적 웹 애플리케이션 MVP입니다.

이 프로토타입은 서버, 데이터베이스, 빌드 프레임워크가 전혀 없는 **순수 정적 웹사이트(Static Website)**로 제작되어, GitHub Pages, Netlify, Cloudflare Pages 등 모든 무료 정적 호스팅 서비스에 즉시 배포할 수 있습니다.

---

## 🎯 MVP 검증 목표
본 MVP 버전 1의 최우선 목적은 다음 가설을 검증하는 것입니다:
> **"학부모가 자녀에게 적합한 주말 지역 체험 프로그램을 10초 이내에 찾을 수 있는가?"**

---

## 🛠️ 기술 스택 (Technical Stack)
기획서의 제약 사항을 철저히 준수하여 다음 기술만을 사용했습니다:
- **HTML5**: 구조화된 시맨틱 마크업 및 접근성(ARIA) 고려
- **CSS3**: 최신 CSS 변수(Custom Properties), Flexbox/Grid 레이아웃, 모바일 퍼스트 반응형 레이아웃, 애니메이션 및 인터랙션 상태 구현
- **Vanilla JavaScript (ES6)**: 라우팅 제어, URL Query Parameter 파싱, 동적 날짜 연산, 로컬 JSON 로드, 실시간 데이터 필터링 및 모달 컨트롤 구현
- **데이터 저장소**: 로컬 JSON 파일 ([programs.json](programs.json))

⚠️ **사용 제한 기술 (금지 사항 준수 완료)**: React, Vue, Angular, Next.js, jQuery, Bootstrap, Tailwind CSS, TypeScript 등 빌드/컴파일 프레임워크와 외부 CSS 라이브러리를 일절 배포하지 않았습니다.

## 💫 핵심 차별화 & 고도화 구현 요소
1. **지속 가능한 주말 날짜 계산기 (Dynamic Weekend Date Calculator)**
   - 기획서의 고정된 날짜 대신, **학부모가 웹사이트에 접속한 실시간 현재 날짜**를 기준으로 가장 가까운 `이번 주말(토/일)`과 `다음 주말(토/일)`의 날짜를 실시간으로 자동 계산합니다.
   - 예: 접속일 기준 가장 가까운 토요일/일요일 날짜를 생성하여 폼 및 추천 카드에 실시간 적용하므로, Mock 데이터가 오래되어 만료되는 현상 없이 **언제나 활성화된 서비스처럼 동작**합니다.
2. **모바일 퍼스트 & 탭 친화적 UI (Tile-Style Pill Selectors)**
   - 자녀 연령대와 관심사를 선택할 때 조작하기 번거로운 기본 라디오 버튼/체크박스 대신, **손가락 터치가 편리한 대형 카드 타일(Tile) 형태의 셀렉터**를 구현했습니다. 
   - 터치 및 포커스 시 부드러운 애니메이션과 포인트 컬러 강조를 통해 높은 접근성과 프리미엄 인터랙션을 선사합니다.
3. **URL 파라미터 기반 정적 상태 연계 (URL Query Parameter Routing)**
   - 단일 페이지 애플리케이션(SPA) 프레임워크 없이도 `find.html`에서 선택한 정보가 URL Parameter(`?age=9-11&interests=Nature&period=this-weekend`)로 안전하게 인코딩되어 `results.html`로 넘어갑니다.
   - 학부모가 특정 필터링 결과를 **즐겨찾기(북마크)에 저장하거나 다른 학부모에게 링크를 카카오톡 등으로 즉시 공유**할 수 있는 강력한 장점을 제공합니다.
4. **인터랙티브 상세 모달 (Popup Overlay Dialog)**
   - 카드의 '신청하기' 클릭 시 정직하지 못한 외부 링크 새 창 연결 대신, **카드 내 상세 소개와 장소, 일정을 깔끔하게 담은 인터랙티브 모달 팝업**을 우선 띄웁니다.
   - 스크롤 락 기능, 이스케이프(ESC) 키 및 바깥 영역 터치 닫기 지원 등으로 네이티브 앱 수준의 조작감을 보장합니다.

## 🚀 로컬 실행 방법 (How to Run)

### 방법 1: 더블클릭 실행
1. 프로젝트 폴더로 이동합니다.
2. `index.html` 파일을 크롬(Chrome), 사파리(Safari), 엣지(Edge) 등 사용하는 웹 브라우저 창으로 드래그 앤 드롭하거나 더블클릭하여 바로 실행합니다.
   - *참고*: 브라우저 보안 정책에 따라 일부 환경에서 로컬 파일 프로토콜(`file://`)로 `fetch()`를 수행할 때 JSON 파일 로드에 CORS 제약이 걸릴 수 있습니다. 이 경우 아래 **방법 2**를 권장합니다.

### 방법 2: 초간단 로컬 개발 서버 구동 (권장)
VS Code의 `Live Server` 확장 프로그램을 실행하거나, 터미널(PowerShell/Command Prompt)을 열고 아래 명령어 중 하나를 입력하여 안전한 로컬 웹 서버를 띄울 수 있습니다.

**Node.js가 설치된 경우:**
```bash
# serve 패키지를 사용하여 서버 구동
npx serve .
```

**Python이 설치된 경우:**
```bash
# Python 3 내장 웹 서버 실행
python -m http.server 8000
```
이후 브라우저에서 `http://localhost:5000` (serve) 또는 `http://localhost:8000` (Python)으로 접속합니다.
